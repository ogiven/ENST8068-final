    • About the data

    This is, in part, dummy data.

    The partner library OCLC symbol code and ILL transaction counts are true FY21-22 data. However, the partner library names, parent organizations, and addresses are fictitious.

    Because the OCLC Library Profiles API is proprietary and only accessible to WorldShare and WorldCat Discovery premium library clients, I was not able query it for the clean, structured library profile data associated with each OCLC code.

    Instead I queried ChatGPT and Google Bard (so I could have two queries running simultaneously) to get stand-in data for the partner library names, parent organizations, and geocodable addresses, associated with the OCLC codes. ChatGPT and Google Bard answered based on their training data from the internet, where OCLC codes are often mentioned with their associated library context. The library profile information is accurate for some of the codes and inaccurate for others.

    A completely accurate map could be made easily by someone with OCLC Library Profiles API access by joining the correct library information to the OCLC codes and remaking the geojson file.

    Data dictionary

    The mapped data includes the following fields

    • Partner_Library: OCLC symbol code for the partner library
    • Latitude: Latitude of the geocoded library address
    • Longitude: Longitude of the geocoded library address
    • Library: Name of the partner library
    • Organization: Parent organization of the partner library
    • City: City where the partner library is located
    • Country: Country where the partner library is located
    • Total_FY21_Activity: Total number of ILL transactions between the partner library and Temple Libraries for fiscal year 2021-2022
    • Total_FY21_LendingToUs: All FY21-22 transactions where the partner library lent something to Temple Libraries
    • Total_FY21_BorrowingFromUs: All FY21-22 transactions where the partner library borrowed something from Temple Libraries
    • Total_FY21_Articles: All FY21-22 transactions between the partner library and Temple Libraries involving electronic articles
    • Total_FY21_Books: All FY21-22 transactions between the partner library and Temple Libraries involving print books
    • Total_FY21_BookChapters: All FY21-22 transactions between the partner library and Temple Libraries involving book chapter scans

    About the maps

    Steps to create the maps

    • Cleaned library address data with OpenRefine to be sure it would geocode
    • Geocoded library addresses using the MMQGIS plug-in for QGIS, which uses Open Street Map
    • Joined the Latitude and Longitude data to the ILL data table on OCLC code in MS Excel and exported CSV file
    • Created GEOJSON data files with Mapbox's geojson.io
    • Created tilesets for the data in Mapbox's Tileset Service using tilesets-cli, following the instructions in this Mapbox tutorial
    • Customized map icons using the Maki icon editor

    The maps use Mapbox vector tilesets and a customized streets style to visualize the data. Copies of the GEOJSON, custom icons, and tilesets-cli recipes used for these are include in the GitHub repository.

    Circle markers and directional lines are data-driven. Circle and line size, and circle marker color intensity represent the transaction volume for the selected partner library.
