Visualizing InterLibrary Loan data

This series of maps visualizes Temple Libraries InterLibrary Loan (ILL) transaction data from fiscal year 2021-2022. InterLibrary Loan is a service that enables patrons of one library to borrow physical materials and receive electronic documents that are held by another library.

Project data is based on the true transaction FY21-22 set, however the partner library addresses are fictitious. See the methodological details for further explanation and a breakdown of the steps needed to create the visualizations.
